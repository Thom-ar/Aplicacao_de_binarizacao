# Aplicacao_de_binarizacao

## Projeto de Aplicação de Escala de Cinza e Binarização de Imagem

Este projeto tem como objetivo aplicar técnicas de processamento de imagem para converter imagens coloridas em escala de cinza e posteriormente aplicar a binarização. A binarização é o processo de transformar uma imagem em preto e branco, onde cada pixel é convertido para 0 (preto) ou 1 (branco) com base em um limiar definido.

### Funcionalidades

- Conversão de imagens coloridas para escala de cinza.
- Aplicação de algoritmos de binarização para transformar a imagem em preto e branco.
- Ajuste do limiar de binarização para obter melhores resultados.

### Estrutura do Projeto

- `Algoritmo de binarizacao.py`: Contém o código para aplicar a binarização nas imagens.
- `README.md`: Este arquivo, que descreve o projeto.

### Como Utilizar

1. Clone o repositório:
    ```
    git clone https://github.com/Thom-ar/Aplicacao_de_binarizacao.git
    ```
2. Navegue até o diretório do projeto:
    ```
    cd Aplicacao_de_binarizacao
    ```
3. Execute o script de binarização:
    ```
    python Algoritmo\ de\ binarizacao.py
    ```
